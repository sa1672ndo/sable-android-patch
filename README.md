# Sable android patch

Allows you to run sable on android by modifying it's `loadLibrary` method to load android native included in this patch mod.

Android sable_rapier native source code can be found here: https://github.com/user01134/sable

# Build instruction

Download the repo, get jdk21 and run ``./gradlew build`` on linux or ``gradlew.bat build`` on windows.
